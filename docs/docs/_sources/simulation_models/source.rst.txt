source
######

The source model constitutes as the input to the simulated Cyber-Physical Process Plant.

Overview
========
The source module consists of an open source component, which is connected via a descending pipe to an output port.
A volume flow sensor can track the volumeFlowRate from the source to the output port.

.. image:: images/source_module.png
  :width: 500
  :alt: source module
**Source module in diagram view**


Functionality
=============
The module provides an input stream of fluids to the process Cyber-Physical Process Plant.


Standard parametrization
========================
There are no variable parameters within this module.
